# taakjs

Taakjs is a Javascript library to provide features of functional programming, DOM manipulation, XHR, streams and JSON.

## Base methods
* extend()
* isLikeArray()
* classOf()
* make()
* partial()
* curry()
* index()
* identity()
* flatten()
* compose()
* dot()
* pick()
* invert()
* eq()
* const()
* anyjoin()
* conjoin()
* disjoin()

## Modifier methods
* not()
* negate()
* recip()
* add()
* sub()
* mul()
* div()
* mod()
* inc()
* scale()
* step()
* shift()
